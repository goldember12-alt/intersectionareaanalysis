# Roadway Graph Figure Inventory and Specifications

**Status: FIGURE AND EXHIBIT PLANNING ONLY.** This document defines candidate exhibits for a future roadway-graph report. It now includes stakeholder-safe aggregate AADT-normalized prototype exhibits. It does not create policy guidance, unit-level rate findings, fixed-band rates, model outputs, regressions, or causal claims.

## Planning Boundary

Candidate exhibits should use the accepted roadway-graph descriptive table package:

- `work/output/roadway_graph/analysis/current/stakeholder_context_table_package/`
- `work/output/roadway_graph/analysis/current/directional_context_descriptive_summaries/`
- `work/output/roadway_graph/analysis/current/signal_context_review_queue/`
- `work/output/roadway_graph/analysis/current/directional_context_distance_band_profiles/`
- `work/output/roadway_graph/analysis/current/signal_direction_context_profiles/`
- `work/output/roadway_graph/analysis/current/descriptive_crash_rate_suppression_review/`

The restored signal-centered report under `docs/reports/signal_centered/` is a style and structure reference only. The roadway-graph report should use the new roadway-derived methodology.

For all exhibits:

- no causal claims
- no unit-level rate findings
- no fixed-band or raw-bin rates
- aggregate AADT-normalized prototype summaries only where suppression-review outputs are used
- no modeling outputs
- no policy guidance language
- review priority is not a danger, risk, or safety-performance ranking
